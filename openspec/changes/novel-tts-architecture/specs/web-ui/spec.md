## ADDED Requirements

### Requirement: FastAPI backend with REST + WebSocket
The Web UI backend SHALL expose RESTful endpoints for CRUD operations and WebSocket endpoints for real-time updates. All endpoints SHALL be documented via OpenAPI (auto-generated by FastAPI).

#### Scenario: Novel upload
- **WHEN** a user uploads a novel file via `POST /api/novels/upload`
- **THEN** the backend SHALL parse the file, create a novel record, and return the novel metadata with a 201 status code

#### Scenario: WebSocket progress stream
- **WHEN** a user connects to `ws://localhost:8000/ws/progress/{novel_id}`
- **THEN** the backend SHALL stream real-time progress events from the pipeline orchestrator

### Requirement: Vue 3 frontend SPA
The frontend SHALL be a Vue 3 + TypeScript SPA with Element Plus components, built with Vite, managing state via Pinia.

#### Scenario: Novel management page
- **WHEN** the user navigates to the novel list page
- **THEN** the page SHALL display all novels with their processing status, chapter count, and action buttons (process, delete, view)

#### Scenario: Character voice editor
- **WHEN** the user clicks "Edit Voices" for a novel
- **THEN** a page SHALL display all characters with their profiles, assigned voices, and a "Preview" button that plays a sample audio

### Requirement: Script review and edit
The frontend SHALL provide a script editor where users can review and modify the AI-generated screenplay before TTS processing.

#### Scenario: Utterance emotion override
- **WHEN** a user changes an utterance's emotion from "neutral" to "angry" in the script editor
- **THEN** the change SHALL be persisted and the TTS for that utterance SHALL be re-queued

### Requirement: Waveform visualization
The frontend SHALL display audio waveforms for chapter previews using WaveSurfer.js, with chapter markers and utterance boundaries.

#### Scenario: Chapter audio preview
- **WHEN** a user clicks play on a completed chapter
- **THEN** the waveform SHALL render with clickable utterance boundaries and synchronized text highlighting
